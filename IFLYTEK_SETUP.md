# 讯飞语音识别API集成说明

## 概述

本项目已集成讯飞（科大讯飞）语音识别API，可大幅提高中文语音识别的准确率。

## 配置步骤

### 1. 获取讯飞API密钥

1. 访问 [讯飞开放平台](https://www.xfyun.cn/)
2. 注册/登录账号
3. 进入控制台，创建新应用
4. 在应用中找到以下三个必要信息：
   - **APPID**: f8a90305（已提供）
   - **APIKey**: 需要填写
   - **APISecret**: 需要填写

### 2. 填写API密钥

打开 `iflytek-api.js` 文件，找到以下部分：

```javascript
const IFLYTEK_CONFIG = {
    APPID: 'f8a90305', // 用户提供的APPID
    APIKey: 'YOUR_API_KEY_HERE', // 需要用户填写
    APISecret: 'YOUR_API_SECRET_HERE', // 需要用户填写
    // ...
};
```

将 `YOUR_API_KEY_HERE` 和 `YOUR_API_SECRET_HERE` 替换为你从讯飞平台获取的实际值。

### 3. 选择API类型

打开网页后，在界面上方可以选择使用哪种语音识别API：

- **浏览器内置API**: 使用浏览器自带的语音识别功能，无需额外配置
- **讯飞语音识别API**: 使用讯飞的API，需要先完成上述配置

## 功能特点

### 讯飞API优势

1. **更高的识别准确率**: 讯飞在中文语音识别领域处于领先地位
2. **支持多种方言**: 除了普通话，还支持多种方言识别
3. **实时结果返回**: 边说边出结果
4. **智能断句**: 自动检测语音结束
5. **动态修正**: 识别过程中会根据上下文动态修正前面的内容

### 当前配置

- **音频格式**: 16kHz采样率，16位深度，单声道PCM
- **识别语言**: 中文普通话（zh_cn）
- **领域**: 通用领域（iat）
- **自动结束**: 检测到语音结束5秒后自动停止
- **最大时长**: 30秒

## 使用说明

1. 完成API密钥配置后，刷新网页
2. 选择"讯飞语音识别API"选项
3. 点击"开始录音"按钮
4. 开始说话，识别结果会实时显示
5. 说完后等待5秒自动停止，或手动点击"停止录音"

- 识别结果返回

## 常见问题

### Q: 提示"讯飞API配置不完整"
A: 请检查是否在 `iflytek-api.js` 中正确填写了 APIKey 和 APISecret

### Q: 连接失败或认证错误
A: 请确保：
   - API密钥填写正确
   - 讯飞应用已开通语音识别服务
   - 网络连接正常

### Q: 识别效果不理想
A: 可以尝试：
   - 检查麦克风是否正常工作
   - 确保环境安静，减少背景噪音
   - 说话清晰，语速适中
   - 在讯飞控制台调整识别参数

### Q: 如何查看API调用次数
A: 登录讯飞开放平台控制台，在应用管理中查看API调用统计

## 技术实现

### WebSocket通信
- 使用WebSocket连接到讯飞服务器
- 实时传输音频数据
- 接收识别结果

### 音频处理
- 使用Web Audio API获取麦克风数据
- 转换为16位PCM格式
- 分段发送音频流

### 认证机制
- 使用HMAC-SHA256生成签名
- 基于时间戳的认证
- Base64编码

## 注意事项

1. **API调用限制**: 讯飞免费版有每日调用次数限制，请合理使用
2. **网络要求**: 需要稳定的网络连接
3. **浏览器兼容性**: 需要支持Web Audio API和WebSocket的现代浏览器
4. **隐私保护**: 语音数据会发送到讯飞服务器进行处理
